# 阿联酋租金回报率计算器

迪拜 / 阿布扎比投资用静态工具：毛回报率、净回报率、回本年限、可选按揭后现金回报率。

**在线访问：** https://SobhaZanYan.github.io/uae-rental-yield-calculator/

## 功能

- **毛回报率** = 年租 / 房价
- **净回报率** = 扣除空置、物业费、托管费、维修预留后的 NOI / 房价
- **含购价外成本净回报**（可选）：配合 [买房成本计算器](https://SobhaZanYan.github.io/uae-buy-cost-calculator/) 填入一次性费用
- **按揭现金回报率**（可选）：等额本息还款后的 Cash-on-Cash
- 中英文切换 · 离线可用
